## 基本套路
1. 判断奇偶性： X & 1 == 1 => 奇数，== 0 => 偶数
2. 把最低位的 1 清 0 => X = X & (X-1)
3. 得到最低位的 1 比如 1100 返回 100 => X & -X
4. 把 X 最右边的 n 位清 0 => X & (~0 << n)
5. 获取 X 的第 n 位值 => (X >> n) & 1
6. 获取 X 的第 n 位的幂值 => X & (1 << (n-1)) ???
7. 仅将第 n 位置为 1 => X | (1 << n)
8. 仅将第 n 位置为 0 => X & (~(1 << n))
9. 将 X 最高位至 n 位（含）清 0 => X & ((1 << n) - 1)
10. 将第 n 位至第 0 位（含）清 0 => X & (~((1 << (n + 1)) - 1))

## 异或

X ^ (~X) = 1(全 1)
X ^ X = 0

a ^ b ^ c = a ^ (b ^ c) = (a ^ b) ^ c
